# Bastion 系统验证检查清单

本文档提供 Bastion 系统的验证点和调试命令，用于在不使用 `runClient` 的情况下验证功能。

## 调试命令

### 基地信息查询
```
/guzhenrenext debug bastion <x> <y> <z>
```
显示指定位置附近的基地信息，包括：
- 基地 ID、状态、转数
- 核心位置、道途类型
- 资源池、污染值、威胁值
- 接管状态

### 基地列表
```
/guzhenrenext debug bastions
```
列出当前维度的所有基地。

## 系统验证点

### 1. 基地创建与扩张
- [ ] 放置核心方块创建基地
- [ ] 菌毯按配置间隔扩张
- [ ] Anchor 在菌毯达到阈值后生成
- [ ] 扩张半径随节点数增长

### 2. 能源节点
- [ ] 能源节点只能放在 Anchor 上方
- [ ] 不同能源类型（光合/汲水/地热/风能/夜能）根据环境生效
- [ ] 能源损耗按距离计算

### 3. 守卫系统
- [ ] 孵化巢在菌毯达到阈值后生成
- [ ] 守卫按配置比例产出
- [ ] 守卫 upkeep 消耗资源池
- [ ] 精英/Boss 按威胁值触发

### 4. 光环效果
- [ ] 玩家进入领域触发压制
- [ ] 隐匿光环给守卫加隐身
- [ ] 侦测光环给玩家加发光
- [ ] 污染阶段影响压制强度

### 5. 防护节点
- [ ] 反爆外壳阻止爆炸破坏
- [ ] 反火外壳阻止火焰蔓延
- [ ] 陷阱节点对范围敌对目标施加减速

### 6. 威胁值系统
- [ ] 拆除节点增加威胁值
- [ ] 威胁值驱动精英/Boss 生成
- [ ] 高威胁值 Boss 有更好奖励

### 7. 污染系统
- [ ] 污染值影响光环强度
- [ ] 污染阶段正确划分（NONE/LIGHT/MEDIUM/CRITICAL）

### 8. 接管流程
- [ ] 净化阵法降低污染
- [ ] 击杀 Boss 进入可接管状态
- [ ] 接管后基地变为友方
- [ ] 友方基地不再施加负面效果

### 9. 友方产出
- [ ] 友方基地定期产出物品
- [ ] 产出物品根据道途不同
- [ ] 产出物品放入附近容器或掉落

## 常见问题排查

### 基地不扩张
1. 检查资源池是否足够
2. 检查扩张冷却是否结束
3. 检查 bastionType 配置的扩张参数

### 守卫不产出
1. 检查孵化巢是否存在
2. 检查资源池是否满足消耗
3. 检查守卫数量是否达到上限

### 接管失败
1. 检查 capturable 状态
2. 检查净化进度
3. 检查玩家权限
