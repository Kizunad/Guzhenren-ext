# Bastion Phase 2 游戏内测试清单

> 本文档为 Phase 2 的 15 个任务提供详细的游戏内手动测试步骤。
> 测试前请确保：`./gradlew runClient` 启动游戏，创建创造模式存档。

---

## 前置准备

### 1. 启动游戏
```bash
./gradlew runClient
```

### 2. 创建测试存档
- 创造模式
- 开启作弊
- 平坦世界（推荐）

### 3. 获取念头资源（占领前提）
```
/guzhenren_debug niantou add 1000
```

### 4. 创建测试基地
```
/guzhenren_debug bastion create ZHI_DAO
```
等待基地扩张到一定规模（约 30-60 秒，观察菌毯蔓延）。

## 命令速查表

```bash
# 基地管理
/guzhenren_debug bastion create <DAO>          # 创建基地（ZHI_DAO/HUN_DAO/MU_DAO/LI_DAO）
/guzhenren_debug bastion info                   # 查看当前位置基地信息
/guzhenren_debug bastion list                   # 列出所有基地

# 传送网络
/guzhenren_debug bastion teleport list          # 列出已占领基地
/guzhenren_debug bastion teleport <bastion_id>  # 传送到指定基地

# 天赋系统
/guzhenren_debug bastion talent info            # 查看天赋点数
/guzhenren_debug bastion talent info <node_id>  # 查看节点详情
/guzhenren_debug bastion talent list            # 列出所有节点
/guzhenren_debug bastion talent convert <amt>   # 资源转换天赋点
/guzhenren_debug bastion talent unlock <node>   # 解锁天赋节点

# 建筑系统
/guzhenren_debug bastion build <blueprint_id>   # 放置建筑蓝图

# 念头管理
/guzhenren_debug niantou add <amount>           # 添加念头
/guzhenren_debug niantou info                   # 查看念头信息
```
